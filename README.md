# 🐾 CatFrens

A Discord bot built around Moosifur — a trading card collector, cat fact dispenser, and seasonal event machine for your server.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🃏 **Card Pulling** | `/random` pulls a Moose photo card with rarity tiers — Common, Rare, Ultra Rare, and Legendary |
| 📸 **Card Creation** | `/grab` and `/grablink` turn Discord images into framed foil trading cards automatically |
| 🗃️ **Collection System** | `/collection`, `/duplicates`, `/wishlist`, `/compare` — collect, trade, and manage your cards |
| 📦 **Booster Packs** | `/daily`, `/weekly`, `/bonuspack` — claim packs on a cooldown with better odds the longer you wait |
| 😺 **Cat Facts** | `/catfact` drops on-demand and on a scheduled interval, with seasonal variants during events |
| 🎃 **Seasonal Events** | Halloween, Winter, Valentine's, Spring, Birthday, New Year's — each with boosted facts and pull odds |
| 🏆 **Leaderboard** | `/topmoose` ranks your server's collectors by completion, card count, or rarest pull |
| 🔥 **Streak System** | Daily claim streaks award bonus packs at prime-number milestones (2, 3, 5, 7, 11…) |
| ⚙️ **Per-Server Config** | `/setup-catfren` lets admins configure fact channels, command restrictions, role cooldowns, and more |
| 👑 **Owner Panel** | `/owner` gives the bot owner a private control panel — gift pulls, gift packs, mass gift an entire server, post Moosifur facts |

## 🚀 Installation

### What you need
- Python 3.10 or newer
- A Discord bot token — [create one here](https://discord.com/developers/applications)
- Your Discord user ID (right-click your name → Copy User ID)

### Run the installer

```bash
python install_catfrens.py
```

Answer 4 questions:
1. Where to install (defaults to a `CatFrens/` folder in the current directory)
2. Your bot token
3. Your Discord user ID
4. A rate-limit log channel ID *(optional — paste a channel ID or press Enter to skip)*

That's it. The installer handles everything else automatically.

### What the installer does

**Installs:**
- `discord.py`, `Pillow`, `aiohttp`
- All bot source files
- 500 cat facts + seasonal event facts pre-loaded
- Card frame assets downloaded from this repo

**Creates:**
- `config.py` with your token and ID already filled in
- `photos/` `rare/` `ultra_rare/` `legendary/` — ready for your Moose photos
- `assets/` — card frame art
- `start_catfrens.bat` — double-click to run, no terminal needed ever again

### After install

1. Drop your Moose photos into the right folders
2. [Invite the bot](https://discord.com/developers/applications) to your server with scopes: `bot` + `applications.commands`, permissions: Send Messages, Attach Files, Read Message History, Use Slash Commands, Manage Messages, Read Members
3. Double-click `start_catfrens.bat` — you're live 🐾

## ⚙️ Configuration

All settings live in `config.py`. Key ones to know:

```python
DEFAULT_PULL_COOLDOWN_MINUTES = 60   # how long between /random pulls
DEFAULT_FACT_INTERVAL_HOURS   = 12   # how often scheduled facts post
PITY_THRESHOLD                = 40   # commons before pity kicks in
RATE_LIMIT_LOG_CHANNEL_ID     = None # channel ID for spam alerts
```

Per-server cooldowns and channel restrictions can be set without editing any files — use `/setup-catfren` in your server.

## 🛠️ Commands

### Everyone
| Command | Description |
|---|---|
| `/random` | Pull a random Moose card |
| `/daily` | Claim your daily booster pack (3 cards) |
| `/weekly` | Claim your weekly booster pack (5 cards) |
| `/bonuspack` | Claim any gifted bonus packs |
| `/catfact` | Get a cat fact |
| `/collection` | View a user's card collection |
| `/profile` | View pull stats |
| `/duplicates` | View, gift, or trade duplicate cards |
| `/wishlist` | Manage your card wishlist |
| `/compare` | Compare your collection with another user |
| `/topmoose` | Server collector leaderboard |
| `/foto` | Post a specific Moose photo by name |
| `/savedfacts` | View facts you've saved with 🐾 reactions |

### Admin
| Command | Description |
|---|---|
| `/setup-catfren` | Configure fact channels, cooldowns, role restrictions |

### Owner only
| Command | Description |
|---|---|
| `/owner` | Private control panel — gift pulls/packs, mass gift, post facts |
| `/grab` `!grab` | Save images from recent channel messages as cards |
| `/grablink` `!grablink` | Save images from a specific message link as cards |
